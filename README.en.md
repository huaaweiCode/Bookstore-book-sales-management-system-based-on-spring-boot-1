# 基于springboot的书店图书销售管理系统1

#### Description
基于springboot的书店图书销售管理系统拥有三个角色

- 管理员：用户管理、角色管理、权限管理、店铺管理等
- 商家：图书管理、上架图书、访问量统计、销售总额统计、订单管理等
- 用户：登录注册、查看图书、购物车、下单、历史订单、结算订单
